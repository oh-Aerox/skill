# 行业板块深度分析框架 (Sector Deep-Dive Framework)
 
## 概述
 
本 skill 指导对任意行业板块进行系统性、多维度的结构化分析，输出兼顾宏观逻辑、中观格局与微观定价的完整研判报告。
 
---
 
## 分析前置：信息侦察
 
在开始分析前，先做以下判断：
 
1. **行业识别与边界厘清** — 用户说的"XX行业"是宽泛概念还是细分赛道？例如"AI"需要拆分为：算力/模型/应用/数据/边缘推理。明确分析边界。
2. **地理范围** — 默认分析中国A股 + 全球视角对比；如用户指定港股/美股/某国，相应调整。
3. **时间视角** — 默认给出短期（6-12个月）、中期（2-3年）、长期（5年以上）三段研判。
---
 
## 分析框架：九大维度
 
按以下顺序逐一分析，每个维度给出**明确结论 + 支撑逻辑 + 风险点**。
 
---
 
### 维度一：产业生命周期定位
 
**目标：判断该行业处于哪个阶段，这决定一切后续逻辑。**
 
使用 S 曲线（Gartner Hype Cycle 变体）定位：
 
```
[萌芽期] → [爆发期] → [泡沫期] → [幻灭期] → [稳定成熟期] → [衰退期]
```
 
分析要素：
- 行业诞生时间与当前渗透率（以全球或目标市场为分母）
- 渗透率加速 or 减速的拐点信号
- 对标历史相似行业（互联网/智能手机/汽车电动化等）的曲线位置类比
- **关键问题**：用户买单行为是否已从"早期尝鲜者"转向"大众市场"？
**输出**：明确标注当前所在阶段 + 预计进入下一阶段的时间窗口。
 
---
 
### 维度二：市场空间 & 饱和程度
 
**目标：量化天花板高度，判断"还有多少肉"。**
 
四象限评估：
 
| 指标 | 分析内容 |
|------|---------|
| TAM（全球总市场） | 当前规模 + 5年预测，引用主流研报/机构数据 |
| SAM（可服务市场） | 行业实际可触达的部分 |
| 当前市场占有率分布 | CR3/CR5 集中度，头部 vs 尾部格局 |
| 饱和度信号 | 价格战烈度、毛利率趋势、新玩家进入速度 |
 
**饱和度判断矩阵**：
- 🟢 未饱和：渗透率 <20%，价格稳中有升，新增需求旺盛
- 🟡 局部饱和：渗透率 30-60%，区域/品类分化明显
- 🔴 高度饱和：渗透率 >70%，价格战为主要竞争手段，行业整合加速
增量来源分析：
- **地理增量**：欠发达市场、下沉市场渗透空间
- **场景增量**：新应用场景解锁（如 AI 进入医疗）
- **人口增量**：老龄化/Z世代等结构性变化带来的需求转移
- **替代增量**：对传统解决方案的替代空间
---
 
### 维度三：高速发展可能性
 
**目标：判断该行业是否存在"爆发"的条件，以及爆发的触发机制。**
 
检查以下催化剂清单：
 
**技术催化剂**
- 是否存在技术奇点（单一突破即改变成本曲线）？
- 学习曲线效应：规模扩张 → 成本下降 → 需求激增的正反馈是否成立？
- 与 AI/自动化/新材料/新能源等通用目的技术的融合点
**政策催化剂**
- 国家战略是否明确支持（如"新基建"、"双碳"、"数字经济"）
- 补贴/税收优惠的力度与持续性
- 监管松绑 or 监管收紧的方向（重要！监管往往是杀手锏）
**需求催化剂**
- 是否存在被压制的刚性需求（政策/价格/技术三重压制解除后的爆发）
- 代际更替：年轻消费群体习惯迁移
- 全球供应链重构带来的国产替代机遇
**资本催化剂**
- 产业资本/国家队/外资流入信号
- IPO/并购活跃度是否加速（往往是景气度领先指标）
**结论评级**：
- ⚡ 高爆发概率（3年内年化增速 >25% 可能）
- 📈 稳健增长（年化 10-25%）
- ➡️ 平稳成熟（个位数增速）
- ⬇️ 收缩整合
---
 
### 维度四：高速增长的可持续性
 
**目标：爆发容易，持续难——判断护城河与增长引擎的耐久度。**
 
**生产力视角分析**（重要！）
 
从以下生产力要素分析增长持续性：
 
1. **人力资本积累**：专业人才培育速度是否跟得上产业扩张？（半导体/航空/核电均受此制约）
2. **资本设备周期**：重资产行业的设备交付周期是否构成供给瓶颈？
3. **原材料/资源约束**：关键原料的地缘政治风险（稀土/锂/硅等）
4. **基础设施匹配度**：电力/算力/物流是否跟得上？（典型：数据中心扩张受电力制约）
5. **技术迭代速度**：技术更新是否导致现有投资快速折旧（技术风险）
**商业模式护城河分析**：
 
| 护城河类型 | 判断依据 | 持久性 |
|-----------|---------|-------|
| 网络效应 | 用户越多价值越大 | ⭐⭐⭐⭐⭐ |
| 转换成本 | 迁移成本/学习成本高 | ⭐⭐⭐⭐ |
| 规模效应 | 单位成本随规模下降 | ⭐⭐⭐ |
| 品牌溢价 | 消费者愿付premium | ⭐⭐⭐ |
| 资源/牌照 | 稀缺许可或原料 | ⭐⭐⭐⭐ |
| 专利技术 | 核心IP壁垒 | ⭐⭐（可被绕过）|
 
**持续性结论**：高速增长预计持续 N 年（给出区间及核心假设）
 
---
 
### 维度五：竞争格局与产业链解构
 
**目标：知道钱最终落在谁口袋里，而不是行业整体好就盲目入场。**
 
**竞争格局分型**：
 
```
完全竞争（商品化）→ 寡头竞争 → 双寡头 → 垄断
     ↓                  ↓           ↓         ↓
  毛利极薄           有结构性强者   胜负已定   监管风险
```
 
**产业链价值分配分析**：
 
画出简化产业链：上游（原料/设备）→ 中游（制造/平台）→ 下游（品牌/渠道/终端）
 
分析：
- 哪个环节定价权最强？（毛利率最高的环节）
- 哪个环节正在发生价值转移？
- 是否存在"微笑曲线"两端集中、中间被压缩的现象？
**中国行业的特殊分析维度**：
- 国央企 vs 民企的份额博弈
- 行业整合是否在"清退尾部"阶段？（有时是最佳买点）
- 出口依赖度与地缘政治敞口
---
 
### 维度六：投资布局空间
 
**目标：给出具体的投资逻辑框架，不是推荐个股，而是教会投资者如何思考。**
 
**布局时机判断**（三段论）：
 
```
行业验证期（高风险高赔率）→ 成长加速期（最佳入场窗口）→ 成熟期（防御性配置）
```
 
**投资布局建议框架**：
 
1. **行业ETF vs 精选个股**：行业弥散度高时ETF更优；龙头格局清晰时精选个股
2. **产业链布局逻辑**：
   - 上游稀缺资源型：景气周期弹性最大，但周期性强
   - 中游核心制造/平台：受益行业成长，但需筛选竞争壁垒
   - 下游品牌/渠道：防御性更强，但弹性较低
3. **分批建仓的时间窗口提示**
4. **需要回避的陷阱**：
   - 假龙头（市值大但护城河弱）
   - 政策依赖型（补贴退坡即利润崩塌）
   - 产能过剩型（扩产过猛导致供给侧塌方）
**资金属性匹配**：
- 激进型：布局萌芽/爆发初期
- 成长型：成长加速期龙头
- 稳健型：成熟期高分红龙头或ETF
---
 
### 维度七：股票市场定价充分性分析
 
**目标：判断市场是否已经"price in"（充分反映）该行业的增长预期。**
 
**定价充分性五项检验**：
 
1. **估值水位检验**
   - 当前板块 PE/PB/PS 与历史分位数对比
   - 与全球同业（美股/港股）横向比较
   - 是否存在"估值溢价"（中国特色赛道往往享受政策溢价）
2. **预期隐含增速测算**
   - 反推：当前股价隐含的未来3-5年增速假设是否合理？
   - 方法：DCF反推或PEG逻辑（PEG > 2 通常意味着定价偏贵）
3. **资金持仓水位**
   - 公募/北向资金的行业配置比例与历史对比
   - 是否已超配（筹码拥挤风险）
4. **分析师覆盖密度**
   - 覆盖机构数量与一致预期离散度
   - 过度一致 = 预期充分 = 上行空间有限
5. **股价与基本面背离信号**
   - 股价领先基本面过多（透支）
   - 股价滞后基本面（仍有补涨空间）
   - 资金流向与股价走势的背离
**定价结论**：
- 🔵 低估/未充分反映：行业景气上行但资金尚未充分入场
- ⚪ 合理定价：股价基本反映当前成长预期
- 🔴 高估/定价过充分：预期透支，需要业绩兑现才能化解

### 维度九：全球视野与比较优势
 
**目标：在全球产业分工中定位中国该行业的角色。**
 
分析要素：
- 中国在该行业全球价值链中的位置（低端制造/核心制造/技术引领/市场为主）
- 全球竞争对手：美国/欧洲/日韩/东南亚在该赛道的布局
- 中国特有优势：供应链完整性、工程师红利、内需规模、政策执行力
- 中国特有劣势：核心技术短板、品牌溢价缺失、地缘政治敞口
- 出海机会：该行业是否具备全球化潜力？

## 分析质量自检清单
 
在输出前，确认：
- [ ] 是否区分了"行业好"和"股票好"？（两者不等价）
- [ ] 是否给出了具体时间窗口而非模糊表述？
- [ ] 是否识别了"最大风险"并给出对应建议？
- [ ] 是否避免了只说"有机会"而不说"什么条件下机会消失"？
- [ ] 国内外数据是否有对比，避免信息茧房？
- [ ] 是否说明了分析的局限性和需要进一步验证的假设？

## 特殊情境处理
 
### 新兴行业（成立 <5 年）
- 重点放在：技术路径验证度、商业模式闭环、资金消耗速度
- 估值方法：优先 PS 或 ARR 倍数，DCF 仅作参考
- 风险提示：流动性风险 > 基本面风险
### 周期性行业（钢铁/煤炭/化工/半导体设备）
- 重点放在：周期位置判断、库存周期、产能利用率
- 估值方法：PB 比 PE 更可靠；用 EV/EBITDA 跨周期对比
- 关键指标：价格/产能/库存三维度同步追踪
### 监管敏感行业（医药/教育/互联网/金融）
- 首先评估政策方向的确定性
- 量化监管风险的"最坏情景"
- 关注监管从严 → 出清 → 龙头受益的转折点
### 出海主导型行业（光伏/储能/新能源车/家电）
- 重点分析目标市场的本地化竞争与关税壁垒
- 汇率风险对利润的敏感度分析
- 地缘政治尾部风险的概率加权
