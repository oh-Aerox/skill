# AI投资估值方法论

本文档提供AI产业链不同环节的估值方法参考。

---

## 一、估值方法选择原则

| 公司类型 | 适用估值方法 | 核心指标 |
|----------|-------------|----------|
| 纯AI算力/芯片 | EV/EBITDA、P/S | 算力产能利用率、毛利率 |
| AI软件/SaaS | EV/NTM Revenue、P/OCF | ARR增长率、NDR、FCF |
| AI基础设施设备 | EV/EBITDA、P/B | 订单book-to-bill、产能利用率 |
| AI应用 | EV/NTM Revenue、DCF | DAU/MAU、变现率、单位经济模型 |
| AI医药/生物 | EV/管线价值、DCF | 临床进度、成功率 |

---

## 二、EV/EBITDA 适用场景

### 适用公司
- 半导体制造和设备（台积电、应用材料、ASML）
- 数据中心基础设施（液冷、电源）
- 重资产AI公司

### 计算要点
```
EV = 市值 + 净债务 + 优先股 + 少数股东权益
EBITDA = 净利 + 利息 + 税 + 折旧 + 摊销
```

### 行业基准
| 环节 | NTM EV/EBITDA 区间 | 备注 |
|------|---------------------|------|
| 先进制程代工 | 15-25x | 周期性强 |
| 成熟制程代工 | 8-15x | 成熟稳定 |
| 半导体设备 | 20-30x | 高壁垒 |
| 数据中心电力 | 12-18x | 公用事业属性 |

---

## 三、EV/NTM Revenue 适用场景

### 适用公司
- AI软件公司（Palantir、C3.ai、Salesforce）
- 尚未盈利的高增长AI公司
- 云计算和AI基础设施

### 计算要点
```
NTM Revenue = 未来12个月预期收入
EV = 市值 + 净债务 + 优先股 + 少数股东权益
```

### 行业基准
| 公司类型 | EV/NTM Revenue 区间 | 合理性判断 |
|----------|---------------------|------------|
| AI基础设施软件 | 15-30x | 成熟高增 |
| AI应用软件 | 8-20x | 取决于增速 |
| 垂直AI SaaS | 5-15x | 依赖渗透率 |
| 概念AI公司 | 30x+ | 高度投机 |

---

 四、P/S (Price/Sales) 适用场景

### 适用公司
- 尚未盈利的AI公司
- 早期AI创业公司
- 用户增长优先于变现的公司

### 计算要点
```
P/S = 总市值 / LTM收入
```

### 参考基准
| 增速 | 合理P/S区间 |
|------|-------------|
| >50% | 15-30x |
| 30-50% | 10-20x |
| 15-30% | 5-15x |
| <15% | 3-8x |

## 五、AI估值特殊考量

### 1. 产能利用率法

对于AI芯片和存储公司，产能利用率是核心指标：
```
合理EV = 产能 × 利用率 × 单位产能EBITDA
```

当产能利用率从70%提升至90%时：
- 固定成本被摊薄
- 毛利率通常非线性提升
- 杠杆效应显著

### 2. 订单 backlog 法

对于设备和材料公司：
```
合理估值 = 已有订单 × 预期毛利率 × 合理PE
```

重点关注：
- book-to-bill ratio >1 表示订单增速 >收入增速
- backlog 增长趋势
- 订单交付周期

### 3. 资本开支周期法

对于AI基础设施：
```
上行周期特征：
- 资本开支指引上调
- 产能利用率提升
- 订单交期延长
- 价格上涨

下行周期特征：
- 资本开支指引下调
- 产能利用率下降
- 库存积压
- 价格下跌
```

### 4. 相对估值法

AI公司往往处于溢价状态，需要相对估值修正：
```
AI溢价倍数 = AI公司P/E / 同类非AI公司P/E

合理溢价倍数：
- 拥有稀缺产能：2-3x
- 拥有技术护城河：1.5-2x
- 纯AI概念：1-1.5x
```

## 六、估值陷阱识别

### 1. 过度关注P/E

AI公司通常研发费用高，P/E失真。优先看：
- EV/Revenue
- EV/EBITDA
- PSR (Price to Sales Ratio)

### 2. 线性外推增长

AI需求往往非线性：
- 初期：需求缓慢释放
- 临界点后：需求爆发式增长
- 泡沫期：预期远超实际

### 3. 忽略产能周期

半导体产能建设周期12-18个月：
- 供不应求阶段：价格上涨、利润暴增
- 产能释放阶段：价格下跌、利润压缩

### 4. 混淆一阶和二阶受益

| 受益层级 | 典型标的 | 估值特征 |
|----------|----------|----------|
| 一阶（直接） | 英伟达 | 高估值、高确定性 |
| 二阶（间接） | 台积电、SK海力士 | 中估值、受益但有时滞 |
| 三阶（传导） | 设备商、材料商 | 低估值、高弹性 |

## 八、估值检查清单

在给出交易建议前，必须检查：

- [ ] 当前估值处于历史区间什么位置？
- [ ] 相对同业是溢价还是折价？
- [ ] 估值是否已经被近期股价涨幅透支？
- [ ] 盈利预测是否已经上调？
- [ ] 催化剂能否支撑当前估值？
- [ ] 有什么风险可能导致估值压缩？
