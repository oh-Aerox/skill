## AI 投资研究分析师

**路径**：`ai-investment-researcher/SKILL.md`

专业 AI 产业投资研究技能，以买方科技研究员视角，通过联网搜索最新公开数据，识别 AI 产业进入下一阶段时的投资转折点，判断哪些可交易资产的需求扩张尚未被市场充分定价。

### 触发场景

- 扫描 AI 投资机会、寻找投资转折点
- 分析 AI 产业链供需变化
- 评估 AI 标的估值和催化剂
- 生成 AI 投资研究报告
- 追踪 AI 资本开支和订单数据
- 比较全球 AI 投资机会（美股 / 港股 / A 股 / 台股 / 日股 / 韩股 / 欧股）

**触发词**：AI 投资、AI 机会、转折点、AI 产业链、AI 赛道、AI 标的、AI 资本开支、AI 需求、AI 供需、AI 产能、AI 订单、AI 催化剂、AI 逻辑、投资扫描、AI Weekly

### 核心能力

- **15 个扫描方向**：训练算力、推理算力、存储和内存、AI 网络、数据中心、端侧设备、软件 Agent、AI 医药、自动驾驶、具身智能、看见世界、感受时间、安全、数据、成本下降
- **七步转折点识别**：新能力 → 新瓶颈 → 付费方 → 供需变化 → 利润池 → 市场预期差 → 交易表达
- **100 分制评分**：需求转折强度、数据证据、利润池质量、未定价程度、催化剂、可交易性、风险可控度
- **多市场覆盖**：美股（最高优先级）、港股、A 股、台股、日股、韩股、欧股
- **可交易资产**：股票、ETF、期权、配对交易、跨市场传导

### 输出内容

- 一页结论表、AI 转折点雷达表
- Top 3–5 交易机会深挖（含逻辑、证据、催化剂、风险、仓位建议）
- 做空/受损资产表、配对交易与跨市场传导表
- 观察名单、数据缺口说明、交易员版本结论

## 行业板块深度分析

**路径**：`sector-analysis/SKILL.md`

对任意行业板块进行系统性、多维度的结构化分析，输出兼顾宏观逻辑、中观格局与微观定价的完整研判报告。

### 触发场景

用户提到某个行业、板块、赛道，或询问「XX 行业值得投资吗」「XX 板块怎么样」「分析一下 XX」「XX 有没有投资机会」时触发。即使用户只是随口问某个行业，也应进行系统性分析。

### 九大分析维度

| 维度 | 核心问题 |
|------|----------|
| 产业生命周期 | 行业处于萌芽 / 爆发 / 泡沫 / 幻灭 / 成熟 / 衰退哪个阶段？ |
| 市场空间与饱和度 | TAM/SAM 规模、渗透率、价格战烈度，还有多少增量？ |
| 高速发展可能性 | 技术、政策、需求、资本四类催化剂是否具备？ |
| 增长可持续性 | 生产力约束与商业模式护城河能支撑多久？ |
| 竞争格局与产业链 | 钱落在谁口袋里？上下游价值如何分配？ |
| 投资布局空间 | 什么时机、什么方式、什么环节布局？ |
| 股票定价充分性 | 市场是否已 price in 增长预期？ |
| 风险矩阵 | 政策、竞争、技术、周期、地缘、估值、黑天鹅 |
| 全球视野 | 中国在全球价值链中的位置与比较优劣势 |

### 输出结构

- 执行摘要与综合评级表
- 九大维度详细分析
- 分投资者类型的行动建议（激进 / 成长 / 稳健）
- 关键假设与风险提示

### 特殊情境

针对新兴行业、周期性行业、监管敏感行业、出海主导型行业，内置差异化的分析重点与估值方法。
